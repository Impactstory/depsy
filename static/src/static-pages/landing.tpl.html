Depsy helps support the software-intensive science of the future by promoting credit for software as a fundamental building block of science.

Credit when software is informally cited

Depsy text-mines OA papers to find fulltext mentions of software they use, revealing impacts invisible to citation indexes like Google Scholar.

Credit when software is reused

Software reuse goes beyond citation—so Depsy analyzes code from over half a million GitHub repositories to find package imports, then uses PageRank to assess the importance of every package in the dependency network.

Credit for all software's authors

Depsy assigns fractional credit to contributors based on designated authorship,
number of commits, and repo ownership—supporting a fairer, more software-native reward system.
